How To Use Gerobak

  1. Get an access and log in
  2. Add a profile and open it
  3. Upload status file
  4. Set software sources
  5. Update repository index
  6. Search, install packages, or upgrade installed softwares
  7. Copy the URLs
  8. Download them using your favorite download manager
  9. Bring the packages home
  10. Install them on your machine

Add a profile

Enter the name, optional description, and architecture of the new profile.

Click the link to open it.

Upload status file

Before able to do anything else, you have to complete three tasks:

  1. Upload the status file of your machine. This file is located at /var/lib/dpkg/status and contains all information about installed packages on your machine.
  2. Set software sources. This is the content of a file called /etc/apt/sources.list on your machine.
  3. Update repository index. Make the repository database up to date by running the update.

Set software sources

In other words, what repositories should be looked at when searching, installing, and upgrading packages.

Update repository index

Download list of packages available in the repositories.

Three new tabs will be made visible after you complete these three tasks.

Search, ...

Now you can search for packages. Mark the result to add it to the to-be-installed-packages list in the Install tab.

Clicking the package name will show you the description of the package.

install new packages, ...

Add some packages to be installed and click the install button after you finish. List of URLs of required packages will be shown.

or upgrade your system

Based on the status file you upload earlier, new packages in the repositories can be checked and downloaded so you can make your system up to date.
